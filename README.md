

# 🍅 Pomo_do_ro Focus Timer

A simple Pomodoro Timer web application built using HTML, CSS, and JavaScript.

This project helps students start focusing with small study sessions instead of feeling pressured to study for long hours from the beginning.

---

## 🌟 Problem Statement

Many students want to focus on their studies, but they think they need to stay focused for a very long time.

Because of this, they:

* feel stressed,
* lose motivation,
* and become disappointed even before starting.

---

## 💡 Solution

This project encourages students to start small and build focus gradually.

Instead of trying to study for many hours, users can:

* start with short focus sessions,
* feel satisfied after completing them,
* and slowly improve their confidence and consistency.

---

* ⏳ Focus Timer
